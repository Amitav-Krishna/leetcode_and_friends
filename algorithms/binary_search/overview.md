# How it works
## Process

1. Pick the element in the middle of our array.
2. Check if that is the search term.
3. Cut the array in half.
4. Repeat until key is found.

## Example

Array: [1, 2, 2, 4, 5, 7, 9, 10, 14, 17, 18].
Key: 10

1a. 7
